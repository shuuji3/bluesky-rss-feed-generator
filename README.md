## Usage

1. Install dependencies:
   ```bash
   pnpm install
   ```

2. Start the server:
   ```bash
   pnpm start
   ```

3. Access the RSS feed:
   `http://localhost:3000/rss/[handle_or_did]`

   Example:
   - `http://localhost:3000/rss/bsky.app`
   - `http://localhost:3000/rss/did:plc:z72i7hdynmk6r22z27h6tvur`
