## Installation

```bash
# Using pnpm dlx (like npx)
pnpm dlx @shuuji3/bsky-rss <command>

# Or install globally
pnpm add -g @shuuji3/bsky-rss
```

## Usage

### Generate RSS (CLI)

Generate an RSS feed for a specific Bluesky user and output it to stdout.

```bash
bsky-rss generate <handle-or-did>
```

Example:
```bash
bsky-rss generate bsky.app > feed.xml
```

### Start Server

Start a proxy server that serves RSS feeds via HTTP.

```bash
bsky-rss serve --port 3000
```

Access the RSS feed at: `http://localhost:3000/rss/[handle_or_did]`

Example:
- `http://localhost:3000/rss/bsky.app`
- `http://localhost:3000/rss/did:plc:z72i7hdynmk6r22z27h6tvur`

## Development

Requires Node.js v25 or later (for native TypeScript support).

```bash
pnpm install
node src/cli.ts generate bsky.app
```
